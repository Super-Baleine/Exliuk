# Usage

## Run the script with Python

```shell
python2.7 passwd.py yourpassword
```

Or

```shell
python2.7 passwd.py
```

## Install it

```shell
chmod +x install.sh;
./install.sh;
```

Or if you want to avoid any error message

```shell
chmod +x install.sh;
./install.sh blah
```


## Run it

```shell
exliuk
```

Or

```shell
exliuk yourpassword
```

Or if you want to copy the generated hash to your clipboard

```shell
exliuk yourpassword blah  
```
